A plugin that automatically disables plugins and/or themes causing certain PHP errors

== Description == 

Click on the "WP Errors" tab, select the errors you'd like to avoid, select any plugins that should be exempted, decide whether theme errors should also be included, and if they should be, pick a back-up theme

Plugin is still in beta, so please report any bugs you might find in the support forum!
